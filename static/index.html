Step 1 — Install OpenClaw

OpenClaw is an open-source AI gateway + TUI chat client. It runs locally with one command.

1

Make sure Node.js 20+ is installed:

Terminal
node --version
2

(Optional but recommended) Install Ollama for local models:

macOS / Windows / Linux
curl -fsSL https://ollama.com/install.sh | sh
3

Install OpenClaw globally via npm:

Terminal
npm install -g openclaw
4

Verify installation:

Terminal
openclaw --version

Step 2 — Add DeepSeek Provider

Configure OpenClaw to use your DeepSeek API key from this platform. Edit the config file:

1

Open your OpenClaw config file:

Location
~/.openclaw/openclaw.json

Full path: C:\Users\YOURNAME\.openclaw\openclaw.json (Windows) or ~/.openclaw/openclaw.json (macOS/Linux)

2

Add this provider config inside the "providers" section. Replace YOUR_API_KEY with the key generated on your API Keys page:

openclaw.json 
{
  "models": {
    "mode": "merge",
    "providers": {
      "deepseek": {
        "baseUrl": "https://deepseek-api-resale-production.up.railway.app/v1",
        "apiKey": "YOUR_API_KEY",
        "api": "openai-completions",
        "models": [
          {
            "id": "deepseek-v4-pro",
            "name": "DeepSeek V4 Pro",
            "contextWindow": 1000000
          },
          {
            "id": "deepseek-v4-flash",
            "name": "DeepSeek V4 Flash",
            "contextWindow": 1000000
          }
        ]
      }
    }
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "deepseek/deepseek-v4-flash",
        "fallbacks": ["deepseek/deepseek-v4-pro"]
      }
    }
  }
}
3

Set the default model for your agents:

openclaw.json
"model": {
  "primary": "deepseek/deepseek-v4-flash"
}

Step 3 — Launch Gateway & Start Chatting

OpenClaw runs a local Gateway that handles all AI requests. You need two terminals:

1

Terminal 1 — Start the Gateway (keep it running):

Terminal 1
openclaw gateway run

Wait until you see "[gateway] ready" — this means everything is working.

2

Terminal 2 — Launch the chat interface:

Terminal 2
openclaw tui
3

That's it! Type your message and press Enter. The AI will respond using your DeepSeek V4 model.

TUI Chat
You: Hello, introduce yourself!

Step 4 — Use API in Your Code

You can also call DeepSeek directly from any OpenAI-compatible SDK. Just point it to our endpoint:

🐍

Python

Python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://deepseek-api-resale-production.up.railway.app/v1"
)

response = client.chat.completions.create(
    model="deepseek-v4-flash",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)
🟩

Node.js

Node.js
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: "YOUR_API_KEY",
  baseURL: "https://deepseek-api-resale-production.up.railway.app/v1",
});

const response = await client.chat.completions.create({
  model: "deepseek-v4-flash",
  messages: [{ role: "user", content: "Hello!" }],
});
console.log(response.choices[0].message.content);
📟

curl (any language)

Terminal
curl https://deepseek-api-resale-production.up.railway.app/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"deepseek-v4-flash","messages":[{"role":"user","content":"Hi"}]}'
