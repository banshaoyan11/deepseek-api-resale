# 快速开始指南

## 立即行动清单

### 第一步：注册必要账户（30 分钟）

1. **DeepSeek API** - 获取 API 密钥
   - 访问：https://platform.deepseek.com/
   - 注册账户（送 500 万 tokens 免费额度）
   - 创建 API Key

2. **Stripe** - 收款解决方案
   - 访问：https://dashboard.stripe.com/
   - 注册商家账户
   - 获取 Publishable Key 和 Secret Key
   - 配置 Webhook URL: `https://yourdomain.com/billing/webhook/stripe`

3. **购买域名** - 建立品牌形象
   - 推荐：Namecheap、Cloudflare
   - 建议域名：`deepapi.com`、`deepsell.com`、`apifast.com`
   - 预算：$10-15/年

### 第二步：配置开发环境（1 小时）

```bash
# 1. 克隆项目
git clone <your-repo-url>
cd DeepSeek_API_Resale_Platform

# 2. 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置环境变量
cp .env.example .env
# 编辑 .env 填入你的配置
```

### 第三步：启动服务（30 分钟）

```bash
# 1. 启动数据库（使用 Docker）
docker run -d \
  --name postgres \
  -e POSTGRES_USER=deepseek \
  -e POSTGRES_PASSWORD=your_password \
  -e POSTGRES_DB=deepseek_api \
  -p 5432:5432 \
  postgres:15

# 2. 启动 Redis
docker run -d \
  --name redis \
  -p 6379:6379 \
  redis:7

# 3. 初始化数据库
alembic upgrade head

# 4. 启动服务
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 第四步：测试流程（1 小时）

1. **注册测试账户**
```bash
curl -X POST http://localhost:8000/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email": "test@example.com", "password": "testpass123"}'
```

2. **登录获取 Token**
```bash
curl -X POST http://localhost:8000/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "test@example.com", "password": "testpass123"}'
```

3. **创建 API Key**
```bash
curl -X POST http://localhost:8000/api-keys/ \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "Test Key"}'
```

4. **测试 API 调用**
```bash
curl -X POST http://localhost:8000/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "deepseek-v4-flash",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

## 常见问题

### Q: 如何获取 DeepSeek API 额度？
A: 注册 platform.deepseek.com，新用户送 500 万 tokens。后续按量充值。

### Q: Stripe 账户审核需要多久？
A: 通常 1-2 个工作日。准备好商业计划书可加快审核。

### Q: 如何防止用户滥用 API？
A:
- 设置速率限制（默认 60 请求/分钟）
- 监控异常使用模式
- 设置每日消费上限
- 实施 IP 黑名单

### Q: 如何处理退款？
A:
- 设置 7 天退款政策
- 在 Stripe Dashboard 中手动处理
- 或实现自动化退款 API
